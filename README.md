# MyOS 🐧

A custom Linux distribution based on **Fedora**, featuring **XFCE Desktop Environment** for a lightweight, efficient, and user-friendly experience.

> **Created by:** JerinHere-Hue at age 11 🚀

## About MyOS

MyOS is a personalized Linux distribution built from Fedora's solid foundation. It combines the reliability and cutting-edge software of Fedora with the lightweight and elegant XFCE desktop environment, making it perfect for both beginners and experienced Linux users.

## ✨ Features

- **Based on Fedora** - Cutting-edge Linux technology and strong community support
- **KDE Plasma Desktop** - Great Desktop Environment And Unmatched Support
- **User-Friendly** - Designed with ease of use in mind
- **Lightweight** - Minimal resource consumption, great for older hardware
- **Highly Customizable** - Tailor the system to your needs
- **Active Community** - Leverage both Fedora and XFCE communities

## 🚀 Getting Started

### System Requirements

- **Processor:** Intel/AMD x86-64 compatible processor
- **RAM:** Minimum 4GB (2GB or more recommended)
- **Storage:** Minimum 10GB free disk space
- **Display:** 1024x768 or higher resolution
